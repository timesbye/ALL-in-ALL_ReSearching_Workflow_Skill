# Product Ideation Example

## Scenario

The user has a rough product idea and wants sharper differentiation instead of a generic AI wrapper.

## Example Prompt

```text
Read the critical-ideation skill first.

Project background:
I want to build an AI copilot for graduate students who manage research papers, experiments, and figures.

Task:
Generate 8 product ideas, critique each one, search for similar tools if current market context matters, then recommend the top 3 directions.

Requirements:
1. Do not just encourage the idea.
2. Call out fake differentiation.
3. Prefer MVPs that can ship in 1-2 weeks.
4. Give a decision matrix and MVP plan.
```

## Expected Shape

- Problem framing with explicit assumptions
- 8 initial ideas
- direct critique per idea
- competitor-aware findings when search is relevant
- narrowed top 3
- one MVP plan per top idea

## Good Outcome Pattern

Strong results usually reject broad "all-in-one research platform" ideas and favor narrower entry points such as:

- a reproducible figure-to-caption workflow
- a reviewer-risk detector for paper drafts
- a literature-to-experiment planning assistant with explicit evidence tracking
