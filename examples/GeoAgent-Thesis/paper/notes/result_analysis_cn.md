# 中文实验分析

从整体趋势来看，GeoAgent 系列相较于通用基线模型在 Accuracy 和 F1 两项指标上均取得了稳定提升，说明该框架在遥感场景理解任务中能够提供更强的判别能力。其中，`GeoAgent-Small` 在延迟仅略高于 `CNN-Baseline` 的情况下，已经实现了明显的性能增益；`GeoAgent-Base` 则在性能与推理开销之间呈现出更均衡的折中；`GeoAgent-Full` 虽然获得了最高的 Accuracy 和 F1，但其额外延迟也最为显著。因此，现有结果更适合支持“GeoAgent 系列存在可利用的性能-延迟权衡”这一结论，而不是直接声称所有变体都在任意部署场景下同时兼顾最优性能与最低成本。这一观察为后续设计预算感知的模型选择或路由机制提供了合理动机。
